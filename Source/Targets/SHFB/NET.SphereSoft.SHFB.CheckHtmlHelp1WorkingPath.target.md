# MSBuild Target SHFB "CheckHtmlHelp1WorkingPath"

This target prevents a build of HTML Help 1 file by SHFB when the working path
contains a `.H` or `.h`.



## Error

- When working path contains a `.H` or `.h` for HTML Help 1 file build
